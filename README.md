# Gym Log 🏋️‍♂️

A premium, modern, and minimalist gym workout tracker built with **Next.js**.

## ✨ Features

- **Two-Column Layout**: 
  - **Weekly Activity**: Scrollable left column with past workout summaries.
  - **Workout Logger**: Robust right column for logging new exercises and sets in bulk.
- **Modular Architecture**: Built with highly reusable components and custom hooks for scalable state management.
- **Premium UI/UX**:
  - Minimalist aesthetic with a black/white/zinc palette.
  - Custom scrollbars and smooth transitions.
  - Fully responsive design (Desktop & Mobile).
  - Modern iconography from **Lucide React**.

## 🛠️ Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **State Management**: React Hooks (State, Props, Custom Hooks)

## 🚀 Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000/home](http://localhost:3000/home) in your browser to see the workout tracker.
